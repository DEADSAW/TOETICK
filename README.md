## üéØ About

TOETICK is a digital implementation of the classic Tic-Tac-Toe game (also known as Noughts and Crosses). This project aims to provide an enjoyable gaming experience with a clean interface and smooth gameplay.

## ‚ú® Features

- **Two Player Mode**: Play against a friend on the same device
- **Player vs Computer**: Challenge the AI opponent
- **Score Tracking**: Keep track of wins, losses, and draws
- **Responsive Design**: Play on desktop, tablet, or mobile devices
- **Interactive UI**: Clean and intuitive user interface
- **Game History**: Review past games and moves
- **Reset Functionality**: Start a new game anytime
- **Win Detection**: Automatic detection of winning combinations
- **Draw Detection**: Identifies when the game ends in a tie

## üéÆ How to Play

1. **Start the Game**: Launch the application to begin
2. **Choose Game Mode**: Select either two-player mode or play against the computer
3. **Make Your Move**: Click on an empty cell to place your mark (X or O)
4. **Take Turns**: Players alternate turns until someone wins or the board is full
5. **Win or Draw**: The game automatically detects winning combinations or draws
6. **Play Again**: Reset the board to start a new game

## üìñ Game Rules

1. The game is played on a 3x3 grid
2. Player 1 uses **X** and Player 2 uses **O** (or computer uses O)
3. Players take turns placing their marks in empty cells
4. The first player to get **3 of their marks in a row** (horizontally, vertically, or diagonally) wins
5. If all 9 cells are filled and no player has won, the game is a **draw**
6. Valid winning combinations:
   - Horizontal: Top row, Middle row, Bottom row
   - Vertical: Left column, Middle column, Right column
   - Diagonal: Top-left to bottom-right, Top-right to bottom-left
